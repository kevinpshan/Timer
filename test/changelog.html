Meeting Tools  ·  Southern Dutchess Toastmasters  ·  All releases from v2.8 onward

v3.1.5
Current test version
Current Feature
New — Welcome Screen
  • Fresh installs now show a proper onboarding screen instead of landing in the app with an auto-created club
  • Three paths: Join Existing Club (enter sync code), Create New Club (with Practice Only option), or Just Exploring (TEST CLUB)
  • Joining by code on the welcome screen pulls roster, fillers, and API key immediately
Fixed — Fresh Install
  • TEST CLUB now seeds correctly on every fresh install — was previously blocked by migration code running first
  • Removed auto-creation of "My Toastmasters Club" which appeared uninvited on every new incognito session
  • v2 migration code now correctly distinguishes a genuine new install from a returning v2 user
Documentation
  • Manual fully rewritten for v3.x — covers all new features, cloud sync, progress tracking
  • Technical reference converted to HTML, updated for multi-club architecture and all new functions
  • Test plan added — 55 test cases across 14 sections with priority levels
  • Changelog added (this document)
v3.1.4
February 2026
Fix
Fixed — Sync
  • Manual Push button now includes API key — previously only roster and fillers were sent
  • Manual Pull now receives and applies API key — Settings field updates live with "Shared by club ☁" placeholder
v3.1.3
February 2026
Feature
New — Practice Only Clubs
  • Clubs can be flagged as Practice Only — local data only, no cloud sync, no Firestore session writes
  • TEST CLUB is automatically marked Practice Only on fresh install
  • Club Manager hides Enable Cloud Sync button for practice clubs, shows ⚗ indicator instead
  • New Club dialog includes Practice Only toggle
  • autoSyncUp() and writeSession() both skip practice clubs automatically
v3.1.2
February 2026
Feature
New — Club-Level API Key Sharing
  • API key moved from device-level to club-level storage in Firestore
  • Key syncs to all club members automatically — one officer sets it up, everyone benefits
  • Settings shows "Shared by club ☁" placeholder when key comes from cloud sync
  • Changing key in Settings immediately pushes to club
New — TEST CLUB Seed
  • Fresh installs automatically create TEST CLUB (Practice) with five sample speakers
  • Sample speakers: Jane Doe, John Smith, Mary Johnson, Robert Brown, Susan Davis
New — Full Name Enforcement
  • Warning modal when adding a single-word name to the roster
  • Explains why full names matter for accurate tracking
  • Two options: go back to fix, or add anyway for edge cases
  • openModal() updated to support custom button labels
v3.1.1
February 2026
Feature
New — Auto-Sync
  • Eliminated manual push/pull for routine operations
  • Auto-pull on app open: silently fetches latest roster and fillers for synced clubs
  • Auto-pull on club switch: immediately pulls data for the newly selected club
  • Auto-push on roster changes: any speaker add or delete triggers instant push
  • Auto-push on filler changes: any filler add, edit, or delete triggers instant push
  • Offline-safe: all auto-sync operations fail silently with no user interruption
  • Manual Push/Pull buttons remain available in Club Manager for force-sync
v3.1.0
February 2026
Major
New — Firebase Cloud Sync
  • Firebase Firestore integration for club roster and filler word synchronization
  • Officers can enable cloud sync from Club Manager — generates unique 8-character sync code
  • Members join by entering sync code — roster, fillers, and settings download immediately
  • Manual Push ↑ and Pull ↓ buttons for controlled sync
  • Session history written to Firestore after every committed session
  • Sync code format: XXXX-XXXX using unambiguous character set (no 0/O/1/I)
New — Progress Tracking
  • Progress screen reads Firestore session history to show speaker improvement over time
  • Overview list with latest filler rate, trend indicator (↑↓→), and session count per speaker
  • Speaker detail view with Latest, Average, and Best filler percentages
  • Session-by-session log with date, role, filler %, word count, and fillers-per-minute
  • Trend calculated by comparing first-half vs second-half of session history
New — Filler Manager
  • Dedicated screen for managing club filler words — add, rename, delete
  • Changes sync to cloud automatically for all club members

v3.0.0
January 2026
Major
New — Multi-Club Architecture
  • Complete data model redesign — all data now scoped to clubs
  • Club Manager screen: create, rename, delete, and switch between clubs
  • Club switcher accessible from main menu header tap
  • All localStorage keys updated to club-scoped format
  • Automatic migration from v2 single-club data to v3 club structure
  • Test and production data remain fully isolated by URL prefix
New — Filler Scoring
  • Filler score calculation after transcription: fillers as percentage of total words
  • Color-coded ratings: Excellent (<5%), Good (5–10%), Needs Work (>10%)
  • Score, word count, total fillers, and fillers-per-minute displayed and saved
