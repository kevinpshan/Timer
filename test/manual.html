1. First Time Setup

When you open Meeting Tools for the first time you will see the Welcome screen with three options:

  • Join Existing Club ‚Äî if your club is already using Meeting Tools, ask your club officer for the 8-character sync code and enter it here. Your roster, filler words, and AI key download automatically.
  • Create New Club ‚Äî enter your club name to set up a fresh installation. You can enable cloud sync later from Settings ‚Üí Clubs.
  • Just Exploring ‚Äî opens the app immediately using the built-in TEST CLUB, pre-loaded with five sample speaker names. No setup required. Nothing you do here affects real club data.
Already a member? Choose Join Existing Club and enter your club's sync code. Everything ‚Äî roster, filler words, and the shared AI key ‚Äî will be ready in seconds.

2. Clubs & Cloud Sync

Meeting Tools supports multiple clubs on one device and keeps shared data in sync automatically.

Switching Clubs

Tap the club name at the top of the main menu (next to the small ‚ñæ) to open the club switcher. A checkmark shows the active club. Switching clubs automatically pulls the latest roster and filler words for that club.

How Auto-Sync Works

Once cloud sync is enabled, the following happen automatically ‚Äî no buttons to press:

  • App opens ‚Äî latest roster and filler words are pulled from the cloud silently.
  • Speaker added or deleted ‚Äî change is pushed to the cloud immediately.
  • Filler word added, edited, or deleted ‚Äî change is pushed to the cloud immediately.
  • Session committed ‚Äî timer or counter result is written to cloud session history for Progress tracking.
  • Offline ‚Äî all sync operations skip silently. The app works fully on local data. Sync resumes next session.

Enabling Cloud Sync (Officers)

  1. Go to Settings ‚Üí Clubs.
  2. Tap Enable Cloud Sync on your club.
  3. An 8-character sync code is generated (e.g. ABCD-EF3G).
  4. Share this code with your members so they can join.

Joining an Existing Club

  1. Go to Settings ‚Üí Clubs ‚Üí Join Existing Club by Code.
  2. Enter the 8-character code from your officer.
  3. Roster, filler words, and AI key sync down immediately.

Practice Club

The built-in TEST CLUB (Practice) is local-only ‚Äî it never syncs to the cloud and never writes to session history. Use it freely for demos and training. You can also mark any new club as Practice Only when creating it via Settings ‚Üí Clubs ‚Üí New Club.

5. Ah-Counter & Filler Words

Track filler words for any speaker in any role during the meeting.

Default Filler Words

Every new club starts with: AH, UH, UM, ER, SO, LIKE, YOU KNOW, plus a Word of the Day placeholder.

Tracking

  • Tap any card to increment the count.
  • Tap the small ‚àí button (bottom right) to decrement.
  • Tap the red ‚úï (bottom left) to remove a filler from this session only.
  • If you save the same speaker twice in one meeting, counts are merged automatically.

Managing Filler Words

Go to Settings ‚Üí Filler Words to add, rename, or permanently delete words for your club. Changes sync to the cloud immediately.

Full Name Requirement

The app encourages full names (First Last) when adding speakers to the roster. This ensures accurate tracking when multiple members share a first name. Single-name entries are allowed but will show a warning.

6. Speech Recording & Transcription

Meeting Tools can record a speaker, transcribe with OpenAI Whisper, and automatically detect filler words ‚Äî no manual counting needed.

How to Use

  1. On the Setup screen in Ah-Counter mode, toggle Record Speech on (appears above the role grid).
  2. Enter the speaker name, select their role, tap START.
  3. On the Transcription screen, tap Start Recording when the speaker begins.
  4. Tap End Speech when they finish. The audio is transcribed automatically.
  5. Tap Merge to Counter to send detected counts to the tally.

Filler Score

After each transcription the app calculates a score based on the percentage of spoken words that were fillers:

  • Under 5% ‚Äî Excellent
  • 5‚Äì10% ‚Äî Good
  • Over 10% ‚Äî Needs Work

Highlights

Filler words appear in red highlight. The active Word of the Day appears in gold highlight.

7. Transcripts

Every transcribed speech is saved for the current meeting so you can share it with the speaker afterward.

  • Access: From the Ah-Counter menu, tap Transcripts.
  • View: Tap any speaker to open their highlighted transcript with filler summary and score.
  • Share: Tap Share to send via email, Messages, Slack, or any app. Format follows your Markdown Export setting.
  • Cleared with meeting data: Transcripts reset when you tap Clear Meeting Data in Settings.

8. Progress Tracking

The Progress screen shows how speakers are improving their filler usage over time across multiple meetings.

Cloud sync required. Progress data is read from Firestore session history. The screen will prompt you to enable cloud sync if it is not yet active.

Progress List

Tap Progress from the Ah-Counter menu. Each speaker shows:

  • Latest filler rate (color coded)
  • Trend indicator ‚Äî ‚Üë Improving, ‚Üì Worsening, or ‚Üí Steady
  • Total number of tracked sessions

Speaker Detail

Tap any speaker to see their full history:

  • Latest, Average, and Best filler percentages
  • Overall improvement trend
  • Session-by-session log with date, role, filler %, word count, and fillers-per-minute

9. Settings

  • Clubs ‚Äî manage clubs, view sync codes, enable cloud sync, push/pull manually, create new clubs, or join by code.
  • Filler Words ‚Äî add, rename, or delete filler words for the active club.
  • Markdown Export ‚Äî when on, reports and transcripts use markdown formatting for Slack and email clients that render it.
  • ChatGPT API Key ‚Äî pre-filled automatically if your club officer set one up. Can be overridden locally.
  • Backup Data ‚Äî exports a JSON file with roster, fillers, club name, and API key as an emergency restore.
  • Restore Data ‚Äî imports a previously exported JSON backup.
  • Clear Meeting Data ‚Äî resets logs, transcripts, and WOD. Does not delete roster or filler words.

10. Data Management

Stored Locally

Speaker roster, filler word list, meeting logs, transcripts (current meeting), WOD selection, app settings.

Stored in the Cloud (sync-enabled clubs)

Speaker roster, filler word list, club API key, and session history (used by Progress screen).

Never Stored in the Cloud

Meeting logs, transcripts, and all practice club data.

11. Setting Up AI (ChatGPT & Whisper)

Club members: If your officer has enabled cloud sync, the API key is shared automatically. You do not need to do anything in this section.

Club Officers ‚Äî Setting the Shared Key

Set it up once ‚Äî it syncs to all members automatically.

Step 1: Go to platform.openai.com and sign up or log in.
Step 2: Navigate to API Keys in the left sidebar.
Step 3: Click + Create new secret key, name it "Toastmasters App", and copy the key (starts with sk-...).
Step 4: In Meeting Tools, go to Settings ‚Üí ChatGPT API Key, paste the key, and tap away from the field. It saves and syncs to the club automatically.

Cost

GPT-4o-mini (W.O.D.) and Whisper (transcription) are extremely inexpensive. A full meeting's worth of usage will typically cost less than $0.05. OpenAI provides free credit to new accounts.
